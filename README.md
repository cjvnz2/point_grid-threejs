## [Three.js] Waves by Value Noise

A Three.js demonstration of dotted waves generated by layering two moving value noise patterns.

![Repo cover image](repo-cover.png?raw=true)
*The output Three.js scene*

## Package versions

- dat.gui: ^0.7.9
- three: 0.146.0
- parcel: ^2.7.0

## How to run locally

1. npm i
2. make dev
